# প্রত্যয় — সংকল্প পাশে থাকার

কুমিল্লা জেলার নাঙ্গলকোট উপজেলায় ২০২০ সালের ২১ ফেব্রুয়ারি প্রতিষ্ঠিত একটি স্বাধীন, সামাজিক, অরাজনৈতিক ও স্বেচ্ছাসেবী মানবতাবাদী সংগঠনের অফিসিয়াল ওয়েবসাইট।

শুধুমাত্র **HTML + CSS + JavaScript** — কোনো বিল্ড টুল বা ফ্রেমওয়ার্ক নেই। GitHub Pages-এ সরাসরি হোস্ট করা যায়।

## ফোল্ডার কাঠামো

```
prottoy-main/
├── index.html          # সম্পূর্ণ ওয়েবসাইট (এক পেজ)
├── style.css           # সব স্টাইল (রঙের সংজ্ঞা :root-এ)
├── script.js           # ডেটা + সব ফাংশনালিটি
├── assets/
│   ├── images/
│   │   └── prottoy-logo.png.png    # লোগো (নাম পরিবর্তন করবেন না)
│   └── docs/
│       └── prottoy-constitution-2nd-edition.pdf
└── .nojekyll           # GitHub Pages-এর জন্য
```

## ফিচার

- **রক্তদাতা তালিকা** — নাম/এলাকা সার্চ + রক্তের গ্রুপ, জেলা, উপজেলা ও উপলভ্যতা ফিল্টার (সব একসাথে কাজ করে)
- **রক্তদাতা নিবন্ধন** — ফর্ম পূরণ করলে নতুন রক্তদাতা তালিকায় যুক্ত হয় (localStorage)
- **রক্তের আবেদন ফর্ম** — সম্পূর্ণ ভ্যালিডেশনসহ
- **দান রসিদ জেনারেটর** — স্বয়ংক্রিয় রসিদ নম্বর ও তারিখ, ঐচ্ছিক QR কোড; PDF ডাউনলোড / প্রিন্ট / ছবি হিসেবে সংরক্ষণ
- **লগইন / নিবন্ধন** — ডেমো অ্যাকাউন্ট সিস্টেম (localStorage)
- **যোগাযোগ ফর্ম**, **নোটিশ বোর্ড**, **গঠনতন্ত্র PDF**
- সম্পূর্ণ রেসপনসিভ (মোবাইল, ট্যাবলেট, ডেস্কটপ) ও `prefers-reduced-motion` সমর্থিত

## তথ্য হালনাগাদ

- **রক্তদাতা যোগ/সম্পাদনা:** `script.js` ফাইলের `DONORS` তালিকা
- **নোটিশ যোগ/সম্পাদনা:** `script.js` ফাইলের `NOTICES` তালিকা

> ⚠️ রক্তদাতার স্পষ্ট সম্মতি ছাড়া ব্যক্তিগত ফোন নম্বর প্রকাশ করবেন না।

## চালানো

কোনো ইনস্টলেশন লাগে না — `index.html` ব্রাউজারে খুললেই চলবে। লোকাল সার্ভার চালাতে চাইলে:

```bash
python -m http.server 8000
# তারপর ব্রাউজারে http://localhost:8000
```

## GitHub Pages-এ ডিপ্লয়

1. রিপোজিটরিতে কোড পুশ করুন
2. **Settings → Pages → Source: main branch** নির্বাচন করুন
3. `.nojekyll` ফাইল আগে থেকেই আছে, তাই আলাদা কনফিগারেশন লাগবে না

## নোট (ডেভেলপারদের জন্য)

- রসিদ জেনারেটরের PDF/ছবি/QR ফিচার CDN থেকে (jsPDF, html2canvas, qrcodejs) প্রয়োজনের মুহূর্তে লোড হয় — মূল পেজ লোডে কোনো প্রভাব ফেলে না
- ব্র্যান্ড রঙ পরিবর্তন করতে হলে `style.css`-এর `:root` ভেরিয়েবল দেখুন (বর্তমান রঙ পরিবর্তন না করার নির্দেশনা রয়েছে)
- লোগো ফাইলের নাম `prottoy-logo.png.png` — এটি ইচ্ছাকৃতভাবে অপরিবর্তিত রাখা হয়েছে
