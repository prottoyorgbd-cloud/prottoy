# ইমেইল সেটআপ গাইড (Email Setup Guide)

## সমস্যা (Problem)
ইমেইল নোটিফিকেশন কাজ করছে না কারণ Gmail কনফিগারেশন সঠিকভাবে সেটআপ করা হয়নি।

## সমাধান (Solution)

### ধাপ ১: Gmail অ্যাকাউন্ট সেটআপ

**গুরুত্বপূর্ণ:** আপনাকে একটি Gmail অ্যাপ্লিকেশন পাসওয়ার্ড তৈরি করতে হবে (নিয়মিত পাসওয়ার্ড নয়)।

1. Gmail অ্যাকাউন্টে লগইন করুন: https://myaccount.google.com
2. বাম মেনু থেকে **"নিরাপত্তা" (Security)** ক্লিক করুন
3. "দ্বি-পদক্ষেপ যাচাইকরণ" (Two-Factor Authentication) সক্ষম করুন (যদি এখনও না করা হয়ে থাকে)
4. আবার সিকিউরিটি পেজে ফিরে যান
5. নিচে স্ক্রল করুন এবং **"অ্যাপ পাসওয়ার্ড" (App Password)** খুঁজুন
6. উপরের ড্রপডাউন থেকে **"মেইল"** নির্বাচন করুন
7. নতুন ড্রপডাউন থেকে **"Windows কম্পিউটার"** (বা আপনার ডিভাইস) নির্বাচন করুন
8. **"তৈরি করুন" (Generate)** বাটনে ক্লিক করুন
9. আপনাকে একটি **১৬-অক্ষরের পাসওয়ার্ড** দেওয়া হবে (স্পেসগুলি উপেক্ষা করুন)

### ধাপ ২: .env ফাইল আপডেট করুন

প্রজেক্টের রুটে `.env` ফাইলটি খুলুন এবং নিম্নলিখিতভাবে আপডেট করুন:

```env
# Gmail Configuration (Gmail থেকে আপনার অ্যাপ পাসওয়ার্ড ব্যবহার করুন)
GMAIL_USER=আপনার-gmail@gmail.com
GMAIL_PASSWORD=আপনার-১৬-অক্ষরের-অ্যাপ-পাসওয়ার্ড

# যেখানে নোটিফিকেশন পাঠাতে হবে
ADMIN_EMAIL=admin@prottoy.com

# সার্ভার পোর্ট
PORT=5000
```

**উদাহরণ:**
```env
GMAIL_USER=prottoy.nk@gmail.com
GMAIL_PASSWORD=abcd efgh ijkl mnop
ADMIN_EMAIL=prottoy.nk@gmail.com
PORT=5000
```

### ধাপ ৩: সার্ভার রিস্টার্ট করুন

টার্মিনালে নিম্নলিখিত কমান্ড চালান:

```bash
npm start
```

বা যদি nodemon ইনস্টল থাকে:

```bash
npm run dev
```

### ধাপ ৪: ইমেইল পরীক্ষা করুন

যেকোনো ফর্ম (রক্তদাতা নিবন্ধন, রক্তের আবেদন, যোগাযোগ) পূরণ করে সাবমিট করুন। আপনার ইমেইলে নোটিফিকেশন আসা উচিত।

## সাধারণ সমস্যা (Common Issues)

### ১. "অনুমতি অস্বীকৃত" (Permission Denied)
- নিশ্চিত করুন যে আপনার অ্যাকাউন্টে দ্বি-পদক্ষেপ যাচাইকরণ সক্ষম আছে
- অ্যাপ পাসওয়ার্ড পুনরায় তৈরি করুন

### ২. "অমান্য শংসাপত্র" (Invalid Credentials)
- `.env` ফাইলে Gmail ঠিকানা সঠিকভাবে লেখা আছে কিনা চেক করুন
- স্পেসগুলি সরান (যদি আপনার অ্যাপ পাসওয়ার্ডে স্পেস থাকে তবুও স্পেস দিয়ে লিখুন)

### ৩: "SMTP সংযোগ ব্যর্থ"
- ইন্টারনেট সংযোগ চেক করুন
- Gmail অ্যাকাউন্ট সক্রিয় আছে কিনা যাচাই করুন

## নতুন বৈশিষ্ট্য

এই আপডেটের সাথে:

✅ **রক্তদাতা নিবন্ধন সিস্টেম আপডেট**
- নিবন্ধন এখন সরাসরি দেখা যাবে না
- এডমিন প্যানেলে "নতুন আবেদন" ট্যাবে যাবে
- এডমিন অনুমোদন করলেই জনসাধারণের কাছে দেখা যাবে

✅ **অনুমোদন ইমেইল**
- রক্তদাতা অনুমোদিত হলে তাকে ইমেইল পাঠানো হবে

✅ **এডমিন প্যানেল উন্নতি**
- দুটি ট্যাব: "অনুমোদিত রক্তদাতা" এবং "নতুন আবেদন"
- পেন্ডিং আবেদনের সংখ্যা দেখা যাবে

## সহায়তা

যদি এখনও সমস্যা থাকে:
1. ব্রাউজার কনসোল খুলুন (F12)
2. ত্রুটি বার্তা নোট করুন
3. সার্ভার লগ চেক করুন (সার্ভার কমান্ড লাইন দেখুন)
